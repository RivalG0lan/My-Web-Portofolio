Project Database Hotel Booking

This project involved building a database using realistic anonymized data sourced from Kaggle, centered on hotel tourism booking records. The dataset was normalized from UNF through 3NF to reduce redundancy and prevent data anomalies. To simplify the implementation process, the data was converted from Excel to Google Sheets before being imported into phpmyadmin via XAMPP.
